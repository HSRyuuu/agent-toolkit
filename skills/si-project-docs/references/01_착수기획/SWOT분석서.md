# SWOT분석서 작성 가이드

## 목적

내부 환경(강점·약점)과 외부 환경(기회·위협)을 체계적으로 분석하여 사업 추진 전략을 도출하고, 시장 진입 리스크와 경쟁 대응 방향을 명확히 하기 위한 문서다.

## 작성 시점

사업기획서·타당성분석서 작성 이후, 구체적인 추진 전략을 수립하는 기획 단계에서 작성한다. 비즈니스모델캔버스와 병행하거나 직전에 작성하면 전략적 일관성을 유지할 수 있다.

## 필수 섹션 구조

### 1. 분석 개요
분석 대상·분석 목적·분석 기준일·분석 방법을 표로 정리.

### 2. SWOT 매트릭스
4개 영역(강점/약점/기회/위협)을 한눈에 볼 수 있는 2×2 아스키 아트 또는 표 형식으로 표현. 각 영역에 ID(S1, W1, O1, T1)를 붙여 이후 섹션과 연결.

### 3. 강점 (Strengths) 상세
ID·강점 요소·상세 내용·중요도(상/중/하)를 표로 정리.

### 4. 약점 (Weaknesses) 상세
ID·약점 요소·상세 내용·개선 방향을 표로 정리.

### 5. 기회 (Opportunities) 상세
ID·기회 요소·상세 내용·활용 방안을 표로 정리.

### 6. 위협 (Threats) 상세
ID·위협 요소·상세 내용·대응 방안을 표로 정리.

### 7. 전략 도출 (SWOT 교차 전략)
- **7.1 SO 전략 (강점 × 기회 — 공격적 성장 전략)**: 전략·연관 요소(Sn × Om)·실행 방안을 표로 정리.
- **7.2 ST 전략 (강점 × 위협 — 경쟁 대응 전략)**: 전략·연관 요소(Sn × Tm)·실행 방안을 표로 정리.
- **7.3 WO 전략 (약점 × 기회 — 약점 보완 전략)**: 전략·연관 요소(Wn × Om)·실행 방안을 표로 정리.
- **7.4 WT 전략 (약점 × 위협 — 리스크 최소화 전략)**: 전략·연관 요소(Wn × Tm)·실행 방안을 표로 정리.

### 변경 이력
버전·변경일·변경 내용·작성자를 표로 관리.

## 작성 팁

- **ID 체계로 교차 전략 연결**: 강점(S), 약점(W), 기회(O), 위협(T)에 번호를 붙이고 교차 전략의 '연관 요소' 열에 "S1 × O2"처럼 명시하면 전략의 논리적 근거가 명확해진다.
- **상세 표에 근거 수치 포함**: 강점·기회 항목은 시장 조사 수치나 내부 측정 지표를 상세 내용에 포함시켜야 주관적 판단이 아닌 데이터 기반 분석임을 입증할 수 있다.
- **교차 전략은 실행 가능한 수준으로**: 전략 도출에서 "차별화 추진" 같은 추상적 표현보다 "X 기능을 Y 채널을 통해 Z 고객에게 제공" 수준의 구체성을 갖추면 후속 기획서 작성 시 바로 활용 가능하다.
